# Sistema de Reserva para Barbearia

Sistema de gerenciamento de reservas para barbearia desenvolvido em Java (console).

## Sobre o Projeto

Este projeto simula um sistema simples de reservas para uma barbearia, onde é possível:

* Cadastrar uma nova reserva
* Escolher um barbeiro
* Verificar se o barbeiro está disponível
* Gerenciar reservas em memória

O objetivo do projeto é praticar conceitos de Programação Orientada a Objetos (POO).
